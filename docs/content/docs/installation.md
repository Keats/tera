# Getting started

To use Tera in your Rust projects, simply add it to your `Config.toml`:

```toml
tera = "0.11"
```

And add the following to your `lib.rs` or `main.rs`:

```rs
#[macro_use]
extern crate tera;
```

You can view everything Tera exports on the [API docs](https://docs.rs/tera).
